# WAVN-Gazebo-Dataset
***RGB &amp; HED Image Collection Script, Image Pairing &amp; CSV File Script***

## Repository Information:
This repository contains the automation script for image and image metadata collection from the robots in the Gazebo simulation environment. It also includes the puthon script for processing, pairing, and labeling the images based upon their metadata. Finally, the README includes information on how to set up the Gazebo environment to emulate that of the experimental setup.

## Instructions To Set-Up Gazebo Environment:
1. Prerequisites. 
   * Ensure you have Ubuntu 22.04, ROS 2 Humble, and Gazebo Harmonic installed.
2. Install dependencies.
3. Clone the repository. Build the packages.
4. Launch the simulation.
   * Include buildings, trees and plants, statues, and vehicles.
